# EDA
[![Build Status][travis-badge]][travis] [![Coverage Status][coveralls-badge]][coveralls]

Pharo Editor with TextMate syntax highlighting support.

## Installation

```smalltalk
Metacello new
	baseline: 'Eda';
	repository: 'github://peteruhnak/eda/repository';
	load
```
